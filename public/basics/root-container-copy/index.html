Root container

The library provides a rendering manager *ebitenui.UI in which you will place your entire interface. It is contains a reference to the root container and responsible for delivering events throughout the user interface.

Here is how to connect it to your game:
 1type Game struct {
 2	ui *ebitenui.UI
 3}
 4
 5func NewGame() *Game {
 6	return &Game{
 7		ui: &ebitenui.UI{},
 8	}
 9}
10
11func (g *Game) Update() error {
12	g.ui.Update()
13	return nil
14}
15
16func (g *Game) Draw(screen *ebiten.Image) {
17	g.ui.Draw(screen)
18}
